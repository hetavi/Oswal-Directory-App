Key Features
Authentication System: Login and registration pages with Firebase integration

Role-Based Access Control: Different views for admin and regular users

Family Management:

Family directory listing

Family detail pages

Forms for adding/editing family information

Request Management:

Committee requests

Member requests

User Management: User role administration

Technical Stack
React (with JSX)

Firebase (Authentication and Firestore)

React Context API (for state management)

Custom hooks

Protected routes

Setup Instructions
Clone the repository

Install dependencies: npm install

Set up Firebase configuration in firebase.js

Run the development server: npm run dev

Available Scripts
npm run dev: Starts the development server

npm run build: Builds the app for production

npm run lint: Runs ESLint for code quality checks
